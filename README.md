# Wushangshu Decision Engine

An AI-powered decision system designed to transform uncertainty into clear, actionable strategies.

## Concept

This project is inspired by Wushangshu, a structured decision framework based on:

- Context (time, environment, actors)
- Positioning (Jian / 間)
- Dynamic conditions (advantage, clarity)

Instead of optimizing actions directly, the system focuses on identifying the correct decision frame before execution.
